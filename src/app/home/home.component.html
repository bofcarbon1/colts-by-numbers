Why Colts By Numbers?

I started this project to combine two areas of interest. Use the Python computer language and track Colts football.
The NFL love stats. Players, teams, offense, defense and rankings. I built the web UI in Angular Material with Flex and responsive design.

The Process Players - Stats - Visuals

We take game stats and store them. Run some cool Python code using AI with statistical math calculations. Take the results and display them in awesome graphs. We share the numbers in Colts fanbase discussions.
